このレポートは日々のランニングのデータを、AIチャットに評価・分析してもらう際の情報を、簡単に、漏れなくまとめるためのhtmlアプリです。
事前の準備としては、Garminを使用している方は、ブラウザ版のGarminコネクトより、評価・分析してもらいたいランニングのログを、CSVデータとしてエクスポートしておきます。
Garminでないデバイスを使用している場合は、csvデータあるいは、Fitデータまたはテキストデータであれば取り込むことができますが、うまくいかない場合は、アプリを更新します。
csv viewer.htmlに上記データをインポートすることで、テキストとして一括コピーを行ってください。
その他には、睡眠時間や安静時心拍数、体調、疲労具合、ランニングの感想や、次の練習予定なども入力できるようになっています。
以上の、事前準備が出来ましたら、index.htmlにアクセスし、用意した事前情報と、テキストデータを貼り付けする等で、レポートが作成できます。
レポートが出来ましたら、コピーをして、AIチャットに貼り付けする準備をします。
入力が不要なものは、とばすこともできますが、デフォルトで設定されているものは、そのままの数値でレポートに記載されますので、チャットに貼り付ける際に削除が必要です。
以降は、使用しているAIチャットに上記でコピーしたテキストを貼り付けして、発信すれば、評価・分析をしてくれます。
